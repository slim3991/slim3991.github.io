Master's Thesis In Progress

"Investigating Robust Tensor Decompositions: MAD-based Censoring and Graph Regularization"

A comparative study on robustifying CP and Tucker decompositions against high-dimensional outliers. The research focuses on the efficacy of iterative outlier censoring—specifically a custom Median Absolute Deviation (MAD) approach—and Graph Laplacian regularization across various anomaly typologies in network traffic data.

FFT and Neural Networks for Identifying and Classifying Heart Arrhythmias

Investigated the viability of using Fast Fourier Transforms to transform ECG signals into frequency components to reduce computational costs in ML-based arrhythmia detection. Compared CNN performance against frequency-domain Feed-Forward NNs.

Featured Projects

Parallelized PCG Solver

C++17 / OpenMP

A high-performance Preconditioned Conjugate Gradient solver for large-scale sparse systems. Features parallelized SpMV kernels using Compressed Sparse Row (CSR) storage and Jacobi preconditioning.

Developed as a high-performance extension for SF2568 (Program Construction in C++).

  • Optimized memory access via custom CSR matrix implementation
  • Scalable performance via OpenMP thread reduction and static scheduling

Incremental SVD Library

C++17 / Eigen / pybind11 / NumPy

A high-performance implementation in C++ of Brand’s Incremental Singular Value Decomposition algorithm for streaming and memory-constrained applications. The library maintains a low-rank approximation of incoming data without recomputing a full SVD.

  • Incremental rank-truncated SVD updates using Brand’s algorithm
  • Supports configurable forgetting factors for adaptive streaming data
  • Python bindings via pybind11 with float32 and float64 support
